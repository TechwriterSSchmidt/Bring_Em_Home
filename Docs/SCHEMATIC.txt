Bring Em Home - Connection Schematic
=====================================
Hardware Platform: SuperMini nRF52840 (Nice!Nano Compatible)
Core: nRF52840 (No Radio/LoRa)

External Components:
- 1.5" OLED Display (SH1107 Driver, 128x128, I2C)
- Bosch BNO085 9-Axis Absolute Orientation Sensor (I2C)
- Quescan M10FD (u-blox M10) GNSS (UART)
- Vibration Motor (Transistor Driver), Buttons, Battery

Connection Diagram:
==================

                                    ┌─────────────────────────────┐
                                    │     SuperMini nRF52840      │
                                    │                             │
    ┌───────────────────────────┐   │ I2C Bus                     │
    │  1.5" OLED (SH1107)       │◄──┤ P0.22 (I2C SDA)             │
    │  (I2C Address 0x3C)       │◄──┤ P0.20 (I2C SCL)             │
    └───────────────────────────┘   │                             │
                                    │                             │
    ┌───────────────────────────┐   │                             │
    │  Bosch BNO085 Compass     │◄──┤ P0.22 (I2C SDA)             │
    │  (I2C Address 0x4A)       │◄──┤ P0.20 (I2C SCL)             │
    │  (Pull-ups required!)     │   │                             │
    └───────────────────────────┘   │                             │
                                    │                             │
                                    │ GPS Module (Serial):        │
                                    │  - RX: P0.08 (Conn to GPS TX)
                                    │  - TX: P0.06 (Conn to GPS RX)
                                    │                             │
                                    │ User Interface:             │
                                    │  - Button 1: P0.15 (Main)   │
                                    │  - Button 2: P0.24 (Aux)    │
                                    │                             │
                                    │ Feedback:                   │
                                    │  - VIB Motor: P0.13         │
                                    │  - RGB LED: P0.17 (Internal)│
                                    │                             │
                                    │ Power Management:           │
                                    │  - Battery ADC: P0.04       │
                                    │  - Charging Stat: P0.05     │
                                    └─────────────────────────────┘

GPIO Pin Summary:
=======================================
Pin Name     | nRF Pin | Function
-------------|---------|------------------------
PIN_I2C_SDA  | P0.22   | I2C Data (OLED + IMU)
PIN_I2C_SCL  | P0.20   | I2C Clock (OLED + IMU)
PIN_GPS_RX   | P0.08   | GPS UART RX (Connects to GPS Module TX)
PIN_GPS_TX   | P0.06   | GPS UART TX (Connects to GPS Module RX)
PIN_BUTTON_1 | P0.15   | Main Action Button (Active Low)
PIN_BUTTON_2 | P0.24   | Secondary Button (Active Low)
PIN_NEOPIXEL | P0.17   | Internal WS2812 LED Data
PIN_VIB_MOTOR| P0.13   | Vibration Motor Control (Active High)
PIN_BAT_VOLT | P0.04   | Battery Voltage ADC (AIN2)
PIN_BAT_CHG  | P0.05   | Charging Status Input
